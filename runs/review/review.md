# 文献综述整理草稿（按大类/小类分组)


## 轴承故障诊断的深度特征学习方法


### 多尺度与多域特征提取

“多尺度与多域特征提取”这一小类主要汇总了 5 篇相关工作，时间范围集中在 2025 年左右。

- 章伟忠等人{基于改进卷积神经网络的滚动轴承故障诊断方法研究}针对滚动轴承在强噪声、单源信息不足和样本稀少工况下故障诊断精度低的问题，提出基于改进卷积神经网络（如POSGMD-ICNN、多传感器融合MSDAENet和小样本原型网络）的系列诊断方法，通过增强特征提取、数据融合和注意力机制，显著提升了诊断精度、鲁棒性和泛化能力。
- 胡怡帆等人{基于深度学习的滚动轴承故障诊断与剩余寿命预测研究}针对滚动轴承故障诊断中特征提取不充分和剩余寿命预测精度低的问题，提出基于双通道特征提取、多尺度残差卷积自编码器及BiGRU与自注意力机制相结合的方法，有效提升了故障诊断准确率与剩余寿命预测精度。
- 毋毓斌等人{基于概率度量与高斯混合模型的轴承故障诊断方法}针对轴承故障数据维度高、特征复杂导致传统方法难以有效提取全局与局部特征的问题，提出结合高斯混合模型与概率度量的改进局部线性嵌入算法，通过融合全局分布规律与局部流形特征，显著提升了故障诊断准确率至95%以上。
- Lei Fu等人{Cross layer-scale fault diagnosis with unsupervised criteria for maximizing information and minimizing redundancy}针对轴承故障诊断中现有方法难以同时捕捉局部脉冲特征、表征冲击性与循环平稳性、以及完整提取多频带故障成分的问题，提出跨层尺度重构方案，通过自适应斜率边缘滤波器等技术，实现了无监督准则下最大化信息与最小化冗余的故障特征提取，显著提升了诊断的准确性和完整性。
- 陈莹洁等人{基于深度学习的滚动轴承故障诊断系统研究与实现}针对工业环境中轴承故障诊断面临强噪声干扰和跨工况数据适应性差的问题，提出基于多尺度特征融合的MSA-Bi GRU模型和结合通道注意力与对抗域适应的FFT-Light DANN方法，开发了前后端分离的在线诊断工具，显著提升了故障诊断准确率至98%以上，为工业设备智能维护提供了高效解决方案。

综合来看，“多尺度与多域特征提取”这一小类的工作主要围绕若干关键问题展开，相关研究大致分布在 2025 年。代表性的研究包括 章伟忠、胡怡帆、毋毓斌 等人的工作，这些方法在公开数据集上验证了有效性，为后续在真实工程场景中的推广应用奠定了基础，但在跨工况泛化能力、可解释性以及与实际工业流程的深度融合方面仍有进一步提升空间。

综合来看，“注意力机制与时序建模”这一小类的工作主要围绕若干关键问题展开，相关研究大致分布在 2025 年。代表性的研究包括 麻全 等人的工作，这些方法在公开数据集上验证了有效性，为后续在真实工程场景中的推广应用奠定了基础，但在跨工况泛化能力、可解释性以及与实际工业流程的深度融合方面仍有进一步提升空间。


### 轻量化与噪声鲁棒特征学习

“轻量化与噪声鲁棒特征学习”这一小类主要汇总了 3 篇相关工作，时间范围集中在 2025 年左右。

- 李璐等人{基于深度学习技术的内燃机车发动机轴承故障诊断方法}针对铁路内燃机车轴承故障诊断中噪声干扰强、工况适应性差、检测效率低的问题，提出融合自适应信号处理与深度学习的智能诊断方法，通过优化振动信号采集与抗干扰特征提取构建轻量化神经网络模型，在混合噪声抑制与多转速域泛化方面取得显著突破，为提升机务段检修效率提供了可靠技术支撑。
- 秦毅等人{面向轴承故障诊断的无监督噪声自适应匹配追踪算法展开去噪网络}针对轴承振动信号噪声导致故障诊断精度受限的问题，提出一种无监督噪声自适应匹配追踪算法展开去噪网络，通过DCT-Laplace字典表示信号成分、小波变换估计噪声并自适应确定网络展开数，显著提高了噪声环境下的诊断精度。
- 朱天伦等人{基于对抗蒸馏与Hessian矩阵的轻量级轴承故障诊断方法}针对传统轴承故障诊断技术效率低、深度学习模型计算成本高的问题，提出了基于对抗蒸馏与Hessian矩阵的轻量级诊断方法，通过知识蒸馏、均匀量化和自动量化感知训练，在显著降低模型存储和计算复杂度的同时保持高诊断准确率，实现了资源受限设备上的高效跨域故障诊断。

综合来看，“轻量化与噪声鲁棒特征学习”这一小类的工作主要围绕若干关键问题展开，相关研究大致分布在 2025 年。代表性的研究包括 李璐、秦毅、朱天伦 等人的工作，这些方法在公开数据集上验证了有效性，为后续在真实工程场景中的推广应用奠定了基础，但在跨工况泛化能力、可解释性以及与实际工业流程的深度融合方面仍有进一步提升空间。

## 跨工况与跨设备的域自适应诊断


### 统计分布对齐方法（MMD/LMMD/子域对齐）

“统计分布对齐方法（MMD/LMMD/子域对齐）”这一小类主要汇总了 29 篇相关工作，时间范围大致覆盖 2023–2025 年。

- Shouqiang Kang等人{Unsupervised fault diagnosis method for rolling bearings based on federated universal domain adaptation}针对滚动轴承私有数据不共享、分布差异和标签空间不一致导致诊断模型精度低的问题，提出基于联邦通用域适应的无监督故障诊断方法，通过随机映射保护隐私、MR-MMD方法对齐特征空间、引入边界向量处理标签差异，实现了跨客户端的有效故障诊断。
- 崔朝阳等人{基于多层联合分布自适应的高速列车轴箱轴承故障诊断方法}针对高速列车轴箱轴承故障诊断中标记数据需求大且需满足独立同分布假设的问题，提出一种基于多层联合分布自适应的无监督模型，采用一维卷积神经网络提取特征并通过多层联合最大均值差异减小跨域分布差异，实现了跨工况下的高效迁移诊断，相比现有方法具有明显优势。
- Xiaoxu Li等人{Research on Unsupervised Domain Adaptive Bearing Fault Diagnosis Method Based on Migration Learning Using MSACNN-IJMMD-DANN}针对变工况下轴承故障诊断中特征提取困难、标记样本获取成本高及域分布差异大的问题，提出基于MSACNN-IJMMD-DANN迁移学习的无监督域自适应方法，通过多尺度注意力机制提取特征、改进联合最大均值差异度量准则减小域差异，并引入对抗损失机制，显著提升了跨域故障诊断的准确性和泛化能力。
- Bin Liu等人{An interpretable physics-informed subdomain moment-enhanced adaptation network for unsupervised transfer fault diagnosis of rolling bearing}针对滚动轴承故障诊断中数据依赖性强、可解释性不足及跨域特征差异大的问题，提出基于物理信息动态卷积和高阶矩度量子域自适应网络的方法，显著提升了诊断过程的可解释性并有效缓解了子域偏移。
- 欧阳炳等人{基于领域自适应的宽度学习滚动轴承故障诊断方法研究}针对滚动轴承故障诊断中特征冗余、参数敏感和跨域泛化弱的问题，提出融合领域自适应与宽度学习的轻量化框架，通过特征选择、参数优化和样本合成协同机制，在提升诊断准确率的同时显著降低计算成本。
- 周圆等人{基于深度迁移学习的滚动轴承故障诊断方法研究}针对滚动轴承故障诊断中传统深度学习依赖大量标记数据且难以处理多尺度特征差异的问题，提出基于纹理损失与核范数正则化的特征增强迁移学习模型，通过多尺度特征对齐和低秩近似提取有效特征，显著提高了诊断精度和鲁棒性。
- 李豪等人{基于联邦学习的轴承故障诊断研究}针对轴承故障诊断中数据隐私保护、样本稀缺、非独立同分布等挑战，提出基于联邦学习的自适应聚合、动态聚类和去中心化改进方法，显著提升了诊断准确率、模型安全性和变工况适应能力。
- 张旭阳等人{基于迁移学习的轮对轴承故障诊断方法研究}针对轮对轴承故障数据稀缺且跨场景分布差异导致智能诊断模型泛化能力不足的问题，提出基于迁移学习的无监督动态对抗联合域适应和结合仿真数据的域泛化方法，显著提升了诊断模型的鲁棒性和跨域泛化能力，并开发了直观的智能诊断系统以支持铁路运维。
- Qi Chang等人{A multi-order moment matching-based unsupervised domain adaptation with application to cross-working condition fault diagnosis of rolling bearings}针对滚动轴承故障诊断中因域漂移导致目标域样本易被误分类的问题，提出一种基于多阶矩匹配的无监督域自适应方法，通过低阶和高阶矩匹配实现粗细粒度混合域对齐，并采用判别性聚类提取域不变特征，显著提升了跨工况故障诊断的准确性。
- 吕智明等人{基于深度域适应的滚动轴承故障诊断方法研究}针对滚动轴承在复杂工况下故障诊断因数据标签稀缺和模型跨域泛化能力不足的问题，提出基于深度域适应的跨工况诊断方法和多源域泛化的未知工况诊断方法，显著提升了模型在不同场景下的诊断准确率和鲁棒性。
- 谭海冰等人{基于改进的多尺度残差网络与迁移学习的轴承故障诊断研究}针对滚动轴承故障诊断中信号噪声干扰、特征提取不充分及跨工况/设备数据分布差异导致性能下降的问题，提出基于改进多尺度残差网络与迁移学习的方法，通过图像转换、特征融合模块优化及分布对齐技术，显著提升了诊断准确性、抗噪性和跨域适应性，降低了标注成本。
- 刘贵毅等人{变转速工况下滚动轴承故障诊断方法研究}针对变转速工况下滚动轴承故障特征难以提取、传统诊断方法失效的问题，提出结合阶次分析与短时傅里叶变换、多尺度卷积与自注意模块、时变迁移学习网络等多种方法，显著提升了诊断的准确性、鲁棒性和跨域泛化能力。
- Shuping Wu等人{KMDSAN: A novel method for cross-domain and unsupervised bearing fault diagnosis}针对跨域无标签轴承故障诊断的挑战，提出基于K-means聚类优化子域对齐的深度子域自适应网络方法，通过改进注意力机制提升特征提取能力，显著提高了诊断精度。
- Hao Luo等人{Normalization-Guided and Gradient-Weighted Unsupervised Domain Adaptation Network for Transfer Diagnosis of Rolling Bearing Faults Under Class Imbalance}针对滚动轴承故障诊断中类别不平衡导致的跨域特征偏移问题，提出一种结合归一化引导和梯度加权的无监督域适应网络，通过增强域不变特征提取和优化损失函数，显著提升了不平衡数据下的故障诊断精度。
- Feng Xiaoliang等人{Unsupervised domain adaptation bearing fault diagnosis method based on joint feature alignment}针对轴承在不同工况下数据分布差异导致跨条件故障诊断效果不佳的问题，提出基于联合特征对齐的无监督域适应方法，通过多层多带宽柯西核最大均值差异和互信息对齐源域与目标域的特征表示，显著提升了跨工况诊断的准确性和泛化能力。
- Bin Liu等人{ISEANet: An interpretable subdomain enhanced adaptive network for unsupervised cross-domain fault diagnosis of rolling bearing}针对滚动轴承无监督跨域故障诊断中噪声干扰导致全局域适应忽略子域分布差异的问题，提出可解释子域增强自适应网络（ISEANet），通过稀疏子段引导降噪层、轻量多特征提取模块等方法增强子域表示，显著提升了跨工况诊断的准确性和可解释性。
